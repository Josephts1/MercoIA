# Disease module for the project MercoIA
This repository contains all the code used for the development of the fruit classification module by disease; the development of this module is part of activity 2 of the PROYECTO DE INVESTIGACIÓN - CONVENIO 268-2025. The objective of the disease classification module is to help farmers in the Santander region identify diseases in pineapple, blackberry, and banana crops using images taken from their own cell phones.
